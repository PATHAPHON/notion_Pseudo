CSS Transform, Transition และ Animation

1. Transform

Transform ใช้สำหรับเปลี่ยนแปลงรูปร่างหรือตำแหน่งของ element โดยไม่กระทบกับ element อื่น

transform: scale(1.5);

Scale: ย่อหรือขยายขนาด (hover เพื่อดูผล)

transform: rotate(45deg);

Rotate: หมุน element

transform: translate(50px, 20px);

Translate: เคลื่อนย้ายตำแหน่ง

transform: skew(20deg);

Skew: เอียง element

2. Transition

Transition ใช้สำหรับกำหนดการเปลี่ยนแปลงให้เกิดขึ้นอย่างนุ่มนวล มี properties หลักดังนี้:

transition-property: background-color, transform;
transition-duration: 0.5s;
transition-timing-function: ease-in-out;

ตัวอย่างการใช้ transition แบบละเอียด (hover เพื่อดูผล)

Properties ของ Transition:
- transition-property: กำหนดคุณสมบัติที่จะให้มีการ transition
- transition-duration: ระยะเวลาในการ transition
- transition-timing-function: รูปแบบการเคลื่อนไหว (ease, linear, ease-in, ease-out)
- transition-delay: ระยะเวลารอก่อนเริ่ม transition

3. Animation

Animation ใช้สำหรับสร้างการเคลื่อนไหวที่ซับซ้อนและต่อเนื่อง โดยใช้ @keyframes

@keyframes bounce {
0%, 100% { transform: translateY(0); }
50% { transform: translateY(-30px); }
}
.animation-bounce {
animation: bounce 1s infinite;
}

ตัวอย่างการเด้งขึ้นลง

@keyframes rotate360 {
from { transform: rotate(0deg); }
to { transform: rotate(360deg); }
}

ตัวอย่างการหมุน

Properties ของ Animation:
- animation-name: ชื่อของ @keyframes
- animation-duration: ระยะเวลาในการเล่น animation
- animation-timing-function: รูปแบบการเคลื่อนไหว
- animation-delay: ระยะเวลารอก่อนเริ่ม
- animation-iteration-count: จำนวนรอบที่เล่น (infinite คือเล่นไม่จบ)
- animation-direction: ทิศทางการเล่น (normal, reverse, alternate)
- animation-fill-mode: สถานะเริ่มต้นและจบของ animation
