CSS Selector Combinators (Parent Selectors)

1. Descendant Selector (Space)

ใช้เลือก element ทั้งหมดที่อยู่ภายใต้ element แม่ ไม่ว่าจะอยู่ลึกแค่ไหนก็ตาม

parent descendant { ... }
ข้อความนี้จะเป็นสีแดง
ข้อความนี้ก็จะเป็นสีแดงเช่นกัน

2. Child Selector (>)

ใช้เลือกเฉพาะ element ที่เป็นลูกโดยตรงของ element แม่เท่านั้น

parent > child { ... }
ข้อความนี้จะเป็นสีน้ำเงิน (ลูกโดยตรง)
ข้อความนี้จะไม่เป็นสีน้ำเงิน (ไม่ใช่ลูกโดยตรง)

3. Adjacent Sibling Selector (+)

ใช้เลือก element ที่อยู่ถัดจาก element ที่ระบุทันที

element + adjacent { ... }

หัวข้อ

ข้อความนี้จะเป็นสีเขียว (อยู่ถัดจาก h3 ทันที)

ข้อความนี้จะไม่เป็นสีเขียว

4. General Sibling Selector (~)

ใช้เลือก element ทั้งหมดที่อยู่ในระดับเดียวกันและอยู่หลัง element ที่ระบุ

element ~ siblings { ... }

หัวข้อ

ข้อความนี้จะมีพื้นหลังสีส้มอ่อน

ข้อความในdiv

ข้อความนี้ก็จะมีพื้นหลังสีส้มอ่อนเช่นกัน
